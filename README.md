# Know Your Rights | ICE Emergency Clarity Tool

## About This Project

This project was built to address a real problem: during ICE encounters, people often panic and don’t know what their rights are or whether their documents protect them. In high-stress moments, it can be difficult to quickly find clear and reliable information.

Our goal was to create a tool that provides simple, step-by-step guidance so users can better understand their situation and respond calmly.

## Problem

During ICE encounters, people may not know:
- Whether officers can enter their home
- What documents actually prove legal status
- What they are required to say or do

This confusion increases fear and vulnerability. We wanted to design something that makes information easier to understand in the moment it’s needed.

## What It Does

The platform includes:

- An Emergency Mode with clear instructions for situations like “ICE is at my door”
- A Personalized Risk Clarity Tool that explains how different documents affect your situation
- A Practice Mode that lets users walk through common encounter scenarios
- Multilingual support (English, Spanish, Traditional Chinese)
- An accessibility mode for larger text and higher contrast
- A fully client-side design with no data collection

## Technology Used

- React
- Tailwind CSS
- JavaScript
- Browser APIs such as SpeechSynthesis and localStorage
- Built using Cursor as an AI-assisted coding tool

The app runs entirely in the browser and does not use a backend or database.

## Privacy

We intentionally designed this project so that it does not collect, store, or transmit any personal immigration data. All functionality runs locally in the user’s browser.

## Disclaimer

This tool is for general educational purposes only and does not provide legal advice. It does not create an attorney-client relationship. Users should consult a qualified immigration attorney for advice about their specific situation.

## Running the Project Locally

1. Clone the repository  
2. Run `npm install`  
3. Run `npm run dev`
