# Know Your Rights — ICE Emergency Clarity Tool

## Overview

During immigration enforcement encounters, panic and uncertainty can increase vulnerability. This platform provides calm, step-by-step guidance to help individuals quickly understand their rights, what their documents mean, and what actions they can take in high-stress situations.

The goal is to replace confusion with clarity in moments where time and composure matter most.

## Problem Statement

During ICE encounters, people often panic and cannot quickly determine whether their documents protect them or what their next step should be. This lack of immediate clarity increases fear and vulnerability.

## Key Features

- Emergency Mode for immediate, step-by-step guidance
- Personalized Risk Clarity Tool
- Practice Mode simulation for encounter preparation
- Multilingual support (English, Spanish, Traditional Chinese)
- Accessibility mode with high-contrast and larger text options
- Fully client-side architecture with no data collection

## Built With

- React
- Tailwind CSS
- JavaScript (client-side only)
- Browser APIs (SpeechSynthesis, localStorage)
- Developed using Cursor (AI-assisted development environment)

## Privacy

This application does not collect, store, or transmit personal immigration data. All logic runs entirely in the browser. No analytics or tracking scripts are used.

## Disclaimer

This tool provides general educational information and does not constitute legal advice. It does not create an attorney-client relationship. Users should consult a qualified immigration attorney or accredited representative for advice specific to their situation.

## Installation

1. Clone the repository  
2. Run `npm install`  
3. Run `npm run dev`
